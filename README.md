# Odin Kittens

This is a quick practice for a project where I will be setting up Rails app to produce API. A basic drill for pure vanilla RESTful resource. This is guided by the [project in Rails course of The Odin Project](https://www.theodinproject.com/lessons/ruby-on-rails-kittens-api#building-a-kittens-api)

## Side note

It may look weird that I am using an integer for the age of kittens, but the focus of this project is to practice building API. Building some sort of age calculator is just not relevant imo.
